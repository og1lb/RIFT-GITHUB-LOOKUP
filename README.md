# RIFT GitHub Lookup

A fast and lightweight tool for retrieving publicly available GitHub account and repository information.

## Features

- 👤 GitHub User Lookup
- 📦 Repository Information
- ⭐ Stars, Forks & Watchers
- 📅 Account Creation Date
- 📝 Public Bio & Profile Details
- 🌍 Location (if public)
- 🔗 Social Links (if available)
- ⚡ Fast & Lightweight
